## 🚀 Başlangıç

Bu proje ile SQL'e veri taşıyabilirsiniz.

### 🛠️ Gereksinimler

Aşağıdaki yazılımların yüklü olması gerekmektedir:

- Node.js ve npm
- MSSQL 
- sql 2017 kullanılabilir.
